# Cirno's Circuits (V0.5.1)

A random assortment of parts and logic for Logic World (v0.92.1).

![image](MainImage.jpg)

## Components Included:
- Adders
- Dividers
- Multipliers
- Comparators
- BCD to Binary Converter
- Hex to 7-Segment Decoder
- Counters
- Decoders
- Multiplexers 
- Demultiplexers
- Integer Printers
- Float Printers
- Extended Relays
- Sideways AND, OR, XOR, NAND, NOR, XNOR Gates
- Extended D-Latches
- D-Flip Flops
- JK Flip Flop
- SR Latch
- T-Flip Flop
- Single and Dual Read Registers
- Ascii Display ROM
- Edge Detector (Rising, Falling, and Dual)

### Experimental (might be deleted in future releases)

- 6502 Control Unit
- 8-to-256 Decoder* (Unstable, might corrupt save files)
- Synchronous RAM (256B and 1kB)

## Install

Make sure any older version of Cirno's Circuits is removed
To install the mod, Download the ZIP file below. Copy the folder called "Cirno's Circuits" and paste it in your GameData folder
